Paragrafos e quebras de linha

Este é um parágrafo de texto. Ele pode conter várias informações e é usado para organizar o conteúdo de forma clara, use a tag <p>.

Se precisar quebrar
linha no texto, use a tag <br>.

Simbolos especiais:
® - Registro de marca
© - Direitos autorais
™ - Marca registrada
€ - Euro
¢ - Centavo
¥ - Iene
£ - Libra
Δ - Delta
↑ - Seta para cima

Emojis:
🙂 - Slightly Smiling Face
🐒 - Monkey
🏀 - Basketball
